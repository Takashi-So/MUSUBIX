# Online Library Management System - Product Context

## 概要

オンライン図書館管理システムは、図書館の蔵書管理、会員管理、貸出・返却・予約管理を効率化するWebアプリケーションです。

## 目的

1. 図書館員の業務効率化（貸出・返却処理の自動化）
2. 会員のセルフサービス化（オンライン予約・延長）
3. 蔵書データの一元管理と検索性向上
4. 延滞管理の自動化と通知機能

## ステークホルダー

| ステークホルダー | 役割 | 関心事 |
|----------------|------|--------|
| 図書館員 | システム運用者 | 業務効率、操作性 |
| 会員（利用者） | エンドユーザー | 検索性、予約の便利さ |
| 図書館長 | 意思決定者 | 統計・レポート |
| システム管理者 | 技術運用 | 保守性、拡張性 |

## 主要機能

1. **蔵書管理**: 書籍の登録・更新・廃棄
2. **会員管理**: 会員登録・更新・停止
3. **貸出管理**: 貸出・返却・延長
4. **予約管理**: 予約・キャンセル・順番管理
5. **検索機能**: 全文検索・カテゴリ検索
6. **通知機能**: 返却期限・予約通知
7. **統計機能**: 貸出統計・人気書籍

## ドメイン

- **MUSUBIXドメイン**: `library`
- **関連コンポーネント**: BookCatalog, MemberRegistry, LoanService, ReservationQueue
